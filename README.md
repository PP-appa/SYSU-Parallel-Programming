# 并行程序设计与算法 (Parallel Programming and Algorithms)

本仓库包含了“并行程序设计与算法”课程的系列实验代码、测试脚本以及结题报告。旨在由浅入深探索现代处理器架构下的计算瓶颈，从串行编程逐渐过渡到基于 MPI (消息传递接口) 及 OpenMP 等体系的并行与高性能计算（HPC）。

## 目录与导航

### [Lab 0: 环境配置与串行矩阵乘法](./lab0/)
- 基于 C/C++ 与 Python 的串行通用矩阵乘法 (GEMM) 算法基准线搭建。
- 深入探索并验证高速缓存（Cache）空间局部性优化机制（即基于 `i-k-j` 循环顺序的读写连续性改良）。
- 实践验证编译优化标签 (`-O3`) 和基于底层汇编/流水线角度的程序循环级展开（Loop Unrolling）。
- Intel MKL 工业级高性能数学计算库环境调用作为并行提速性能的最终比较标杆。

### [Lab 1: MPI 点对点通信与并行矩阵计算](./lab1/)
- 构建多进程网络架构模型，应用原生的 `MPI_Send` 及 `MPI_Recv` 点对点分布式信息传送机制。
- 学习经典的 **多进程主从调度分配模式 (Master-Worker Model)**。
- 通过数据收集和透视表格，直观分析：在任务并行分配机制下，节点进程数量与矩阵计算任务极大规模变化时，所浮现出的加速比以及因网络通讯造成的额外开销占比分析。
- 探索在极端条件限制（有限内存：外存换页分配与流水线掩盖）和特殊数据场景（高度稀疏数据阵列：CSR 压缩存储、运算剪枝等算法及重新分配平衡权重点）下的理论突破与迭代手段。
